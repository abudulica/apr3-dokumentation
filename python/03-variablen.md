# Variablen

Variablen sind Platzhalter für Werte, die sich während der Programmausführung ändern können.

:::note
Bei der Erstellung einer Variable spricht man von **Deklaration**. Bei der erstmaligen Belegung mit einem Wert spricht man von **Initialisierung**.
:::

Es gibt verschiedene Datentypen:

- Integer (int): Ganzzahlen, zB 23, 12, ...
- String (str): Zeichenketten mit Anführungszeichen, zB "Anna", "Max", etc.
- Kommazahlen (float): zB 23.24, 34.11, ...
- Bool (bool): Wahrheitswerte, zB True, False

In Python wird der Datentyp automatisch erkannt.

:::note
Der Datentyp einer Variable kann sich in Python dynamisch ändern. Dies kann ev. zu einem unvorhergesehenen Ergebnis führen.
:::
