Es gibt sehr viele Bibliotheken für verschiedene Anwendungsgebiete.

## Arbeit mit VS Code

In einem Ordner wird ein **Terminal** (= Eingabeaufforderung) geöffnet und anschließend der Befehl `code .` aufgerufen.

:::tip
Mit einem Rechtsklick auf einen Ordner kann ebenfalls im Kontextmenü das Projekt mit VS Code geöffnet werden.
:::

Mit der **Python-Erweiterung** wird die Programmierung erleichtert (Menü Erweiterungen in VS Code).

Mit <kbd>STRG</kbd>-<kbd>SHIFT</kbd>-<kbd>P</kbd> kann der installierte Python-Interpreter ausgewählt werden.

:::note
Python-Programme haben die Dateierweiterung `.py`.
:::
