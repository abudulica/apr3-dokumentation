# Zeichenkettenfunktionen

Zeichenketten können mit Funktionen bearbeitet werden.

- Länge: `len()`
- Teilzeichenkette: `[start:ende]`
- Suche: `find()`
- Groß- und Kleinschreibung: `upper()`, `lower()`

```python
# zeichenketten.py

text = "Hallo, Python!"

laenge = len(text)
suche = text.find("Python")

print("Länge:", laenge)
print("Position von Python:", suche)
```

:::danger
Zeichenketten beginnen bei der **Position 0**!
:::
