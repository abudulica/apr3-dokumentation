# Projektziele

Ein Projekt verfolgt ein **klares Ziel**. Um Ziele besser überprüfbar zu machen werden **SMARTE** Ziele erstellt.

SMART steht für
- **S:** spezifisch - eine konkrete Formulierung ist notwendig
- **M:** messbar - die Zielerreichung muss überprüfbar sein
- **A:** attraktiv - Die Mitarbeiter müssen die Ziele akzeptieren und sich dafür einsetzen
- **R:** realistisch - die Ziele müssen erreichbar sein
- **T:** terminiert - die Ziele müssen über einen Termin verfügen

**Beispiel:** Die Unit-Test-Abdeckung des Quellcodes der *Example*-Software oll bis zum 30. Juni 2026 von 65% aud 70% des Quellcodes erhöht werden.

**Nicht Ziele:** Als Nicht-Ziel bezeichnet man ein Ziel, welches nicht unmittelbar als Projektziel definiert wurde. Das Ziel könnte durchaus in einem Nachfolgeprojekt verfolgt werden.
